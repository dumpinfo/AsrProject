Trace Output

HCOPY supports the following trace options where each trace flag is given using an octal base

00001
basic progress reporting.
00002
source and target file formats and parameter kinds.
00004
segment boundaries computed from label files.
00010
display memory usage after processing each file.
Trace flags are set using the -T option or the TRACE configuration variable.
