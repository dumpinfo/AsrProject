Whats New In Version 3.3

This   section lists the new features in HTK Version 3.3 compared to the preceding Version 3.3.

  1. HEREST now incorporates the adaptation transform generation that was previously performed in HEADAPT. The range of linear transformations and the ability to combine transforms hierarchically has now been included. The system also now supports adaptive training with contrained MLLR transforms.

  2. Many other smaller changes and bug fixes have been integrated.
