Cut-offs

With a back off scheme low count events can be discarded - cut-off - from the model and more frequently observed shorter-context estimates can be used instead. An additional advantage of discarding low occurrence events is that the model size can be substantially reduced, since in general as decreases so the number of events increases - in fact the Good-Turing discounting scheme depends upon this relationship.
