Word LM file formats

Language models can be stored on disk in three different file formats - text, binary and ultra. The text format is the standard ARPA-MIT formad used to distribute pre-computed language models. The binary format is a proprietary file format which is optimised for flexibility and memory usage. All tools will output models in this format unless instructed otherwise. The ultra LM format is a further development of the binary LM format optimised for fast loading times and small memory footprint. At the same time, models stored in this format cannot be pruned further in terms of size and vocabulary.
