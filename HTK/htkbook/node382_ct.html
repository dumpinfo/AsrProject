Use

LGLIST is invoked by typing the command line 

   LGList [options] wmapfile gramfile ....
The specified gram files are printed to the output. The -grams are printed one per line following a summary of the header information. Each -gram is printed in the form of a list of words followed by the count.

Normally all -grams are printed. However, if either of the options -i or -f are used to add words to a filter list, then only those -grams which include a word in the filter list are printed.

The allowable options to LGLIST are as follows

-f w
Add word w to the filter list. This option can be repeated, it can also be mixed with uses of the -i option.
-i n
Add word with id n to the filter list. This option can be repeated, it can also be mixed with uses of the -f option.
LGLIST also supports the standard options -A, -C, -D, -S, -T, and -V as described in section 4.4.
