Tracing

LPLEX supports the following trace options where each trace flag is given using an octal base

00001
basic progress reporting.
00002
print information after each utterance processed.
00004
display encountered OOVs.
00010
display probability of each -gram looked up.
00020
print each utterance and its perplexity.
Trace flags are set using the -T option or the TRACE configuration variable.
