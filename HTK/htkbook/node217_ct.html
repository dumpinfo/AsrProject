Class LM file formats

Class language models replace the word language model described in section 16.7 with an identical component which models class -grams instead of word -grams. They add to this a second component which includes the deterministic word-to-class mapping with associated word-given-class probabilities, expressed either as counts (which are normalised to probabilities on loading) or as explicit natural log probabilities. These two components are then either combined into a single file or are pointed to with a special link file.
