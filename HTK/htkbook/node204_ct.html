Memory problems

Should you encounter memory problems then try altering the amount of space reserved by the tools using the relevant tool switches such as -a and -b for LGPrep and LGCopy. You could also try turning on memory tracing to see how much memory is used and for what (use the configuration TRACE parameters and the -T option as appropriate. Language models can become very large, however - hundreds of megabytes in size, for example - so it is important to apply cut-offs and/or discounting as appropriate to keep them to a suitable size for your system.
